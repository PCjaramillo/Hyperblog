Git

  • Nuevas ramas
  • git init
  • git add .
  • git commit -m
  • git status
  • git log o log --stat
  • git branch cabecera(Nombre)
  • git checkout entrar a la cabecera
  • git merge master, para fucionar los dos proyectos

    • GitHub/p>

  • git remote add origin pegar enlace de github
  • git push origin master enviar al repositorio
  • git pull origin master
  • git pull origin master --allow-unrelated-histoies fucionar las ramas

    • Llaves shh

  • Crear llave shh-keygen -t rsa -b 4096 -C "Correo electronico"
  • Ver si el servidor shh esta corriendo eval $(ssh-agent -s)
  • agregar la llave ssh-add ~/.ssh/id_.rsa
  • Cambir url de origin (git remote set-url origin pegas la url)

    • Rutas para ver las ramas

  • crear una ruta (alias ruta="git log --all --graph --decorate --oneline"

    • Tag

  • git tag -a V0.1 -m "Resultado de las clases" pones la ruta
  • (Saber a que commit esta conectado el tag) git show-ref --tags
  • (agregar tag) git push origin --tags
  • (Borrar un tag)git tag -d nombre del tag
